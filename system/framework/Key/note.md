
##Directory Structure

###Abstracts
Abstract classes for database, controller and model, etc.

###Cache
Cache implementation, such as Redis.

### Database
Database implementation, such as Mysql.

### Exception
Exception classes.

###Handlers
Error handlers.

###Http
Base http implementation, such as Request, Response, etc.

###Inputs
Input implementation, such as String, Array and so on.

###Interfaces
Interface classes.

###Log
Log implementation.

###Middleware
Middleware classes, such as Router, Session and so on.

###Records
Input extension, complex input type.

###Util
Defines the useful tool classes
